# CRM System — Управление клиентами

Веб-сервис для управления клиентами с возможностью просмотра, поиска, фильтрации и обновления (редактирования) данных.

---

## Стек технологий

| Категория | Технология |
|-----------|------------|
| Backend | Java 21, Spring Boot 3.2.6 |
| База данных | PostgreSQL |
| Миграции | Liquibase |
| Шаблонизатор | Thymeleaf |
| Безопасность | Spring Security |
| Сборка | Maven |
| Тестирование | JUnit 5, Mockito, Spring Boot Test |

---

## Функциональность

### Базовая функциональность

- Регистрация и аутентификация пользователей
- Просмотр списка клиентов с пагинацией
- Отображение краткой информации о клиенте (ФИО, телефон, компания, текущий статус) в общем списке
- Просмотр карточки клиента с полной информацией
- Обновление данных клиента, включая изменение статуса
- Получение списка доступных статусов через серверное API
- Отображение статуса обработки запроса (успех / ошибка)
- Сохранение изменений в данных клиента с валидацией

### Дополнительная функциональность

- **Поиск клиентов по имени** - поиск по имени или фамилии
- **Поиск клиентов по компании** — поиск с автодополнением
- **Фильтрация клиентов по статусам** — множественный выбор статусов через чекбоксы
- **Подсветка статусов в UI** — цветовая индикация статусов клиентов

> **Примечание:** пользовательский интерфейс реализован полностью на английском языке.

## Пререквизиты

Перед запуском необходимо развернуть базу данных **PostgreSQL** со следующими параметрами:

| Параметр | Значение |
|----------|----------|
| База данных | `crm_db` |
| Пользователь | `postgres` |
| Пароль | `242368` |
| Порт | `5432` |

Схема базы данных и начальные данные (справочники статусов, регионов, компаний) создаются автоматически при старте приложения через **Liquibase** миграции.

## Инструкция по запуску

### Способ 1. Сборка и запуск через Maven

```bash
# Клонировать репозиторий
git clone https://github.com/USERNAME/crm-coursework.git
cd crm-coursework

# Сборка проекта
mvn clean install

# Запуск приложения
java -jar target/crm-coursework-1.0.0-SNAPSHOT.jar
```

### Способ 2. Запуск через bat-файл (Windows)

Bat-скрипт и jar-файл должны быть скачаны и размещены в одной папке. В корне этой папки выполнить команду:

```batch
run.bat
```

Путь к JDK, указанный в скрипте: `C:\Program Files\Eclipse Adoptium\jdk-21.0.6.7-hotspot` (папка по умолчанию при установке JDK c adoptium.net).

### Способ 3. Запуск через shell-скрипт (macOS / Linux)

В корне папки проекта выполнить команду:

```bash
./run.sh
```

Путь к JDK, указанный в скрипте: `/Library/Java/JavaVirtualMachines/temurin-21.jdk/Contents/Home`.
Путь к jar-файлу, указанный в скрипте: `target/crm-coursework-1.0.0-SNAPSHOT.jar`.

### Способ 4. Запуск через Spring Boot Maven Plugin

В корне папки проекта выполнить команду:

```bash
mvn spring-boot:run
```

**После запуска приложение доступно по адресу: [http://localhost:8080](http://localhost:8080)**

## Наполнение тестовыми данными

В репозитории размещен SQL-скрипт `src/main/resources/init-500-clients.sql` для наполнения таблицы клиентов **500 тестовыми записями**.

**Важно:** выполните скрипт **после запуска приложения** (чтобы Liquibase создал таблицы).

```bash
# Выполнить скрипт в PostgreSQL
psql -U postgres -d crm_db -f data/insert_500_clients.sql
```

Или через pgAdmin / DBeaver: открыть файл и выполнить.
